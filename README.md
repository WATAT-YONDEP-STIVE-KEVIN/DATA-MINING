# Data Mining – Implémentations Python

Ce dépôt réunit des implémentations Python de techniques de data mining et de clustering, avec une attention particulière sur les méthodes basées sur les graphes et sur des algorithmes classiques de clustering.

## Objectifs
- Fournir des implémentations claires et documentées
- Servir de base pédagogique pour des TP et des expériences pratiques
- Faciliter la comparaison entre différentes approches de clustering

## Contenu

- **Clustering par graphe**
  - **METHIS**: L'algorithme de partitionnement parallèle de graphes multiniveau METIS est une approche puissante pour diviser un graphe en sous-graphes tout en minimisant le nombre de connexions inter-sous-graphes. Il est souvent utilisé dans des applications comme la simulation numérique, le calcul scientifique, et l'optimisation.
- **LCM**: Mining Low Cost High Utility
Itemsets
- **K-means**: implémentation du k-means et variantes utilisées
- **CHA**: Classification hierachique
- **GSP**
- **TP pratique**: exercices pratiques pour appliquer les algorithmes sur des datasets simples
